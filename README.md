MODULO BASICO DIDACTICO DE ODOO PARA ALUMNOS Y MODULOS
Se trata de un módulo personalizado para Odoo en donde podemos ver dos modelos: alumnos y modulos.
En el módulo personalizado los alumnos se matriculan en diferentes módulos en un instituto posible instituto. 
Se práctica la relación entre modelos y en concreto en este caso es una relación Many2many.
